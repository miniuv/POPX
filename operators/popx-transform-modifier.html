POPX Transform Modifier

Overview

The POPX Transform Modifier applies translation, rotation, and scaling transformations to incoming particles or instances. It supports both world space and local space transformations and integrates seamlessly with the popxOrient attribute for orientation-aware motion.

This operator is essential for procedural animation, particle manipulation, and instance-based effects in TouchDesigner. All parameters are animatable and CHOP-drivable for dynamic control.

Parameter Pages

Transform Page

Name Type Description
Translate X / Y / Z Float Offset along corresponding axes. Moves particles or instances in world or local space.
Rotate X / Y / Z Float (degrees) Rotation around each axis in degrees. Applied in XYZ order.
Scale X / Y / Z Float Non-uniform scaling along each axis. Default value is 1.0 for no scaling.
Uniform Scale Float Uniform scaling multiplier applied equally to all axes. Multiplies with per-axis scale values.
Pivot XYZ Defines the pivot point for rotation and scaling operations. Defaults to origin (0, 0, 0).
Space Menu Choose between Local or World space for transformations. Local space respects particle orientation.

Orientation Page

Name Type Description
Affect Orientation Toggle When enabled, updates the popxOrient quaternion attribute to reflect rotational changes.
Affect Velocity Toggle When enabled, applies rotational transformations to the velocity attribute.
Use Delta Transform Toggle Enables relative (delta) mode for accumulated transforms. Useful for time-based procedural motion.

Common Page

Name Type Description
Enable Toggle Enables or disables the operator. When disabled, input data passes through unchanged.

Inputs

  • Input 0: Particle or instance stream with position data.

Outputs

  • Output 0: Transformed particle or instance stream with updated attributes.

Attributes Affected

  • P — Position (vector3)
  • v — Velocity (vector3, if Affect Velocity is enabled)
  • popxOrient — Quaternion orientation (vector4, if Affect Orientation is enabled)
  • scale — Scale (vector3 or float)

Notes

  • Combine with POPX Noise or POPX Magnetize for dynamic procedural effects.
  • Delta transforms allow for time-accumulated motion without resetting state.
  • All transform parameters support animation and external CHOP control.
  • Local space transforms respect the particle's popxOrient quaternion if present.
  • When working with instances, use this operator before geometry instancing for best results.
