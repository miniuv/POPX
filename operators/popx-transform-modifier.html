POPX Transform Modifier

POPX Transform Modifier

The POPX Transform Modifier applies procedural transformations — translation, rotation, and scaling — to incoming particles or instances. It can operate in both world space and local space, and supports orientation-aware transforms via the popxOrient attribute, enabling MOPS-style workflows.

Parameters

NameTypeDescription
Translate X / Y / ZFloatApplies translation offsets along the corresponding axes.
Rotate X / Y / ZFloat (°)Rotates particles or instances around the specified axes (degrees).
Scale X / Y / ZFloatScales particles along each axis independently.
Uniform ScaleFloatApplies a uniform scale multiplier to all axes.
PivotXYZDefines the pivot position for rotation and scaling operations.
SpaceMenuLocal / World — where the transform is applied.
Affect OrientationToggleIf enabled, rotations modify the popxOrient quaternion attribute.
Affect VelocityToggleIf enabled, transformations apply to the velocity attribute for consistency.
Use Delta TransformToggleEnables relative (delta) mode to accumulate changes over time (MOPS-style).
EnableToggleEnables/disables the operator without removing it from the network.

Inputs

  • Input 0: Particle or instance stream.

Outputs

  • Output 0: Transformed stream with updated position, orientation, and scale.

Attributes Affected

  • P — Position
  • v — Velocity (if Affect Velocity is enabled)
  • popxOrient — Quaternion orientation (if Affect Orientation is enabled)
  • scale — Non-uniform scale vector or uniform scale float

Notes

  • Works well when chained with POPX Noise, POPX Magnetize, or POPX Instancer.
  • Delta transforms allow procedural rigging or progressive adjustments across frames.
  • All parameters can be driven by CHOP channels for interactive control.
