POPX Transform Modifier

The POPX Transform Modifier applies translation, rotation, and scaling to incoming particles or instances. It supports both world space and local space transformations and integrates with the popxOrient attribute for orientation-aware motion.

Parameters

NameTypeDescription
Translate X / Y / ZFloatOffset along corresponding axes.
Rotate X / Y / ZFloat (°)Rotation around each axis in degrees.
Scale X / Y / ZFloatNon-uniform scaling along axes.
Uniform ScaleFloatUniform scaling multiplier.
PivotXYZDefines pivot point for transforms.
SpaceMenuChoose Local or World space.
Affect OrientationToggleUpdates popxOrient quaternion.
Affect VelocityToggleApplies transformations to velocity.
Use Delta TransformToggleEnables relative (delta) mode for accumulated transforms.
EnableToggleEnables/disables the modifier.

Inputs

  • Input 0: Particle or instance stream.

Outputs

  • Output 0: Transformed stream with updated attributes.

Attributes Affected

  • P — Position
  • v — Velocity
  • popxOrient — Quaternion orientation
  • scale — Scale vector or float

Notes

  • Combine with POPX Noise or POPX Magnetize for dynamic effects.
  • Delta transforms allow time-accumulated procedural motion.
  • All parameters are animatable and CHOP-drivable.
