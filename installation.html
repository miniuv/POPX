Installation & Getting Started

The POPX toolkit is distributed as a collection of .tox components that can be imported into any TouchDesigner project.

Steps:

  1. Download or clone the POPX repository from GitHub.
  2. Extract the folder and locate the /popx directory containing all operators.
  3. In TouchDesigner, drag and drop the desired .tox file (e.g., POPX_Transform_Modifier.tox) into your network.
  4. Each operator comes pre-wired with standard POPX attributes and can be chained with others.
  5. For updates, simply replace the component in your project with the latest version.

System Requirements

  • TouchDesigner 2023.11+ (Experimental builds recommended)
  • GPU with OpenGL 4.6+ and compute shader support
  • Windows 10/11 or macOS 13+

Recommended Structure

/popx
├── generators/
├── modifiers/
├── simulations/
├── tools/
└── followoffs/
