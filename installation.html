Getting Started with POPX

Follow this guide to install and configure POPX operators in your TouchDesigner environment.

Download POPX

POPX is distributed as a collection of .tox component files organized by operator type. Download the latest release from one of the following sources:

Download Latest Release View on GitHub

Latest Version: v1.0.0 (Released: January 2025)
File Size: ~45 MB
License: MIT License

Installation

Step 1: Extract the Archive

After downloading, extract the POPX.zip archive to a location of your choice. The recommended location is: 

C:\Users\[YourName]\Documents\TouchDesigner\POPX\

Or on macOS/Linux:

/Users/[YourName]/Documents/TouchDesigner/POPX/

Step 2: Folder Structure

After extraction, your POPX folder should contain the following structure: 

/POPX
  ├── generators/
  │   ├── popx_scatter.tox
  │   ├── popx_source.tox
  │   └── ...
  ├── modifiers/
  │   ├── popx_transform_modifier.tox
  │   ├── popx_noise_modifier.tox
  │   └── ...
  ├── falloffs/
  │   ├── popx_falloff_field.tox
  │   └── ...
  ├── tools/
  │   ├── popx_voxelize.tox
  │   └── ...
  ├── simulations/
  │   ├── popx_flip_solver.tox
  │   └── ...
  └── README.txt

Step 3: Add to TouchDesigner Palette (Optional)

To add POPX operators to your TouchDesigner component palette for easy access:

  1. Open TouchDesigner
  2. Navigate to Edit → Preferences
  3. Select the Palette tab
  4. Click Add Folder
  5. Browse to your POPX installation directory
  6. Click OK to save

POPX operators will now appear in your component palette, organized by type.

Step 4: Using POPX Operators

To use a POPX operator in your network:

  1. In the TouchDesigner network editor, press Tab to open the operator menu
  2. Type the operator name (e.g., "popx scatter")
  3. Select the operator from the list
  4. Alternatively, drag and drop .tox files directly from your file browser into the network

Verify Installation

To verify that POPX is installed correctly:

  1. Create a new TouchDesigner project
  2. Add a POPX Scatter operator to your network
  3. Connect a Box SOP to the first input of POPX Scatter
  4. Create a Geometry COMP and connect POPX Scatter to it
  5. You should see particles scattered on the box surface

If particles appear, your installation is successful!

Your First POPX Network

Let's create a simple particle system to get familiar with POPX:

Basic Particle Scatter with Transform

Box SOP
  ↓
POPX Scatter (emit 1000 particles)
  ↓
POPX Transform Modifier (add rotation)
  ↓
Geometry COMP (render as instances)

Detailed Steps:

  1. Create source geometry: Add a Box SOP
  2. Scatter particles: Add POPX Scatter, set "Total Count" to 1000
  3. Add motion: Add POPX Transform Modifier, set "Rotate Y" to 45
  4. Render: Create Geometry COMP, enable "Instancing"

Troubleshooting

Operators Not Appearing in Palette

  • Verify the palette path in Edit → Preferences → Palette
  • Restart TouchDesigner after adding the palette folder
  • Check that .tox files are not corrupted (re-download if necessary)

GPU Compute Errors

  • Update your graphics drivers to the latest version
  • Verify OpenGL 4.6 support with glxinfo (Linux) or GPU-Z (Windows)
  • Reduce particle count if experiencing out-of-memory errors

Performance Issues

  • Enable "Cook on Demand" for operators not actively being used
  • Reduce particle counts during initial setup, scale up for final renders
  • Use falloff operators to limit modifier influence regions
  • Monitor VRAM usage in TouchDesigner's performance monitor
