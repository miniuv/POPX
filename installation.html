Installation

Download POPX

POPX is available exclusively through Patreon. To access it, subscribe to the POPX Tier and download the latest release.

Subscribe on Patreon Watch Installation Guide

Latest Version: v1.0.0
Release Date: November 2025

Each release includes the POPX operators package and an example project file to help you get started quickly.

System Requirements

TouchDesigner: 2025 Experimental (2025.xxx) or newer — required to access the new POPs (Point Operators) family.

GPU: NVIDIA or AMD GPU with support for Vulkan 1.1 / OpenGL 4.5.

Operating System:

  • Windows 10/11 — full support for all GPU features.
  • macOS Monterey (12) or newer — functional, but with limited GPU feature support.

GPU Memory: Minimum 4 GB VRAM, 8 GB+ recommended for large simulations.

⚠️ Note: Some POPX operators use GPU atomic operations, which may not be fully supported on macOS or integrated GPUs.

Installation Steps

  • Subscribe to the POPX Tier on Patreon and download the POPX.tox package.
  • Open your TouchDesigner project.
  • Drag the POPX.tox file directly into your network.
  • Your project may freeze momentarily while the system initializes — this is normal.
  • Once loaded, the POPX container will appear at the root level of your project.
  • Press Tab to open the OP Create Dialog — you'll now see the new POPX operator family available for use.

Verify Installation

To verify that POPX is installed correctly:

  • Create a new TouchDesigner project.
  • Add a POPX Scatter operator to your network.
  • Connect a Box SOP to the first input of POPX Scatter.
  • Create a Geometry COMP and connect POPX Scatter to it.
  • You should see particles scattered on the box surface.
