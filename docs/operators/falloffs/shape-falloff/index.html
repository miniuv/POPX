Shape Falloff

Summary

The Shape Falloff generates a Falloff Attribute based on signed distance fields (SDFs) computed from various geometric primitives. Internally, it uses the Field POP to calculate distance-based influence regions with smooth gradients and precise transitions.

It provides accurate control over falloff distribution using mathematical shape definitions such as spheres, boxes, cylinders, and planes, with adjustable parameters for size, radius, height, and roundness.

This approach ensures continuous, easily controllable falloff values that can be refined through transition ranges and remapping options.
