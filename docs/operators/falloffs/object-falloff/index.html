Object Falloff CorePro

Summary

The Object Falloff creates falloff patterns based on the spatial relationship between points and arbitrary 3D geometry objects. This allows for complex, art-directable falloff patterns driven by modeled shapes rather than mathematical primitives.

Supports multiple area of influence modes including inside/outside detection, surface distance calculations, and surface intersection proximity. Objects can be transformed in 3D space with full control over translation, rotation, scaling, and pivot points, with configurable transform and rotation orders.

The resulting falloff values can be combined with existing falloff attributes, enhanced with procedural noise, remapped with custom curves, and previewed with color ramps.
