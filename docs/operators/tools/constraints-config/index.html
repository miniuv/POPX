Constraints Config

Summary

Constraints Config is the final pre-processing step before the Soft Body simulation. It takes all constraint geometry generated by the Constraints operators and converts/transfers it onto the simulated geometry points in a compact structure. This configuration is required so the Soft Body knows what constraints to solve.

Parameters

Page: Config

Soft Body Solver Solver – Reference to the Soft Body simulation operator that will use these constraints.

Page: Common

Free Extra GPU Memory Freeextragpumem – Free memory that has accumulated when output memory has grown and shrunk.

Inputs

Input 0 POP – Geometry
Input 1 POP – Constraints
Input 2 POP – Collision Geometry

Outputs

Output 0 POP – Geometry
Output 1 POP – Constraints
Output 2 POP – Collision Geometry
Constraints Delete
