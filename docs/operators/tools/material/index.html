Material

Summary

Material is a utility operator that applies Disney BRDF material attributes to geometry for use with the Path Tracer operator. This operator writes material properties as primitive attributes, allowing you to create custom per-primitive materials instead of using the Path Tracer's built-in global material. The Material operator provides the same comprehensive Disney principled BRDF parameters as the Path Tracer, including base color, metallic/roughness workflow, specular properties, subsurface scattering, sheen, clearcoat, transmission, and emission.

By using the Material operator, you can assign different material properties to different parts of your geometry, enabling multi-material scenes for path traced rendering. The operator supports material override control, allowing you to choose whether to replace existing material attributes or preserve them. All material parameters follow the Disney BRDF standard, ensuring physically accurate rendering with intuitive artist-friendly controls.

Material outputs geometry with material attributes written to primitive data, ready to be consumed by the Path Tracer. This workflow separates material definition from the rendering process, allowing you to build complex material libraries and apply them flexibly across your geometry. When used with the Path Tracer's "Override if Material Exists" toggle disabled, the Path Tracer will read and respect these per-primitive material attributes instead of using its global material settings.

Specular Tint Speculartint – Tint specular reflection with base color (0 = white, 1 = colored).
Anisotropic Anisotropic – Anisotropic highlight stretching (0 = isotropic, 1 = fully anisotropic).
Anisotropic Rotation Anisotropicrotation – Rotation angle for anisotropic highlights (0-1 = 0-360 degrees).
Surface Surface – Header for surface scattering properties.
Subsurface Subsurface – Subsurface scattering amount for translucent materials like skin or wax.
Sheen Sheenh – Header for fabric sheen properties.
Sheen Sheen – Soft velvet-like reflection for fabrics and cloth.
Sheen Tint Sheentint – Tint sheen with base color (0 = white, 1 = colored).
Clearcoat Clearcoath – Header for clearcoat layer properties.
Clearcoat Clearcoat – Secondary specular layer for car paint or lacquered surfaces.
Clearcoat Gloss Clearcoatgloss – Glossiness of clearcoat layer (0 = rough, 1 = glossy).
Advanced Advanced – Header for advanced material properties.
IOR Ior – Index of refraction for dielectrics (1.0 = air, 1.45 = plastic, 1.5 = glass).
Transmission Transmission – Light transmission through surface for glass and transparent materials.
Emission Emissionh – Header for emissive light properties.
Emission Emission – Emissive color for self-illuminating surfaces (RGB, values > 1 = bright lights).
