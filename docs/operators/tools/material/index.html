Material

Summary

Material is a utility operator that applies Disney BRDF material attributes to geometry for use with the Path Tracer operator. This operator writes material properties as point attributes, allowing you to create custom per-point materials instead of using the Path Tracer's built-in global material. The Material operator provides the same comprehensive Disney principled BRDF parameters as the Path Tracer, including base color, metallic/roughness workflow, specular properties, subsurface scattering, sheen, clearcoat, transmission, and emission.

By using the Material operator, you can assign different material properties to different parts of your geometry, enabling multi-material scenes for path traced rendering. The operator supports material override control, allowing you to choose whether to replace existing material attributes or preserve them. All material parameters follow the Disney BRDF standard, ensuring physically accurate rendering with intuitive artist-friendly controls.
