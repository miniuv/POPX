Delete

Summary

Delete removes points from POPX Geometry based on various selection criteria, functioning like TouchDesigner's Delete POP but specifically designed for POPX Geometry structure. This operator provides multiple methods for selecting which points to delete including attribute tests, pattern matching, group membership, bounding volume tests, and thinning operations.

The operator offers flexible selection through sequences of rules that can be combined with AND/OR logic. You can delete points based on attribute values (using comparison functions), point number patterns, group assignments, or spatial bounding tests. Additionally, the Thin page provides pattern-based deletion using step intervals and random probabilities for creating sparse point distributions.

Each selection method can be inverted to keep the selected points and delete everything else, and multiple rules can be combined to create complex selection criteria. This makes Delete essential for culling unwanted geometry, creating variations through thinning, or isolating specific subsets of your POPX Geometry for further processing.
