Visualize Frame

Summary

Visualize Frame is a handy tool for quickly visualizing the local orientation axes of any packed primitive. This operator creates visual representations of coordinate frames, making it easy to debug and understand the orientation of geometry in your scene.

The operator can visualize the orientations of packed primitives, as well as any other meshes or points that have a p@orient attribute, or v@N and v@up attributes. This makes it an invaluable debugging tool for understanding how transformations and rotations are applied to geometry.

The Length Scale parameter allows you to adjust the size of the visualization axes to match the scale of your geometry, ensuring the frame visualization is clearly visible and proportional to your scene.

Parameters

Page: Visualize Frame

Length Scale Lengthscale – Controls the length of the orientation axis visualization lines.

Page: Common

Free Extra GPU Memory Freeextragpumem – Free memory that has accumulated when output memory has grown and shrunk.
Render Primitives Renderprimitives – Toggles rendering of POPX Geometry or shows it as point instances only.
SRT / RST Srtrst – Sets the transform order when using POPX Geometry as built-in TouchDesigner instances.

Inputs

Input 0 POP – POPX Geometry or geometry with orientation attributes to visualize

Outputs

Output 0 POP – Geometry with visualized orientation frame axes
