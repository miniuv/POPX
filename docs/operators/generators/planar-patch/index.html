Planar Patch v1.3.0

Summary

Planar Patch generates a planar triangulated mesh from a variety of shapes. It produces well-distributed triangles with a target edge length and optional relaxation iterations to improve mesh quality. Supported shapes include Rectangle, Circle, Trapezoid, and Ring, each with shape-specific controls for corners, taper, skew, inner size, and arc angles.
