Relax

Summary

The Relax operator applies a relaxation algorithm to spread instances apart and prevent overlaps. It uses iterative solving to push overlapping instances away from each other, creating more evenly distributed spacing.

You can optionally provide constraint geometry (surface) or a constraint volume (3D texture) to keep instances from flying off into space. The operator supports two solver modes: Simple for immediate single-frame relaxation, and Advect for progressive time-dependent solving.

Relax supports two methods: Neighbor (searches for nearby instances within a radius) and Field (uses a voxel-based field approach). The operator considers the PointScale attribute of each instance as its radius for overlap detection. If there is no PointScale attribute, the radius defaults to half of the search radius. Optional falloff-based modulation enables spatially varying relaxation effects.
