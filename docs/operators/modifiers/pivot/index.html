Pivot

Summary

The Pivot operator adjusts the pivot points of instances relative to their individual bounding boxes. This operator allows you to move pivots either as a transformation of the instance itself, or as an invisible pivot-only adjustment.

You can use this operator to ensure that the pivots of objects of different sizes and shapes are all scaling or rotating from the same relative positions. This is particularly useful when you need consistent transformation behavior across heterogeneous instance collections.

The operator supports multiple modes: Center Pivot (centers the pivot), Align to BBox (aligns to bounding box sides), Shift Pivot (offsets by a specific amount), Set Pivot (Local) (sets an explicit local-space position), and Set Pivot (World) (sets an explicit world-space position). Optional falloff-based modulation enables spatially varying pivot adjustments.
