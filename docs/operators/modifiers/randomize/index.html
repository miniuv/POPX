Randomize

Summary

The Randomize operator adds randomness to instance transforms including position, rotation, and scale. Each randomization parameter can be controlled independently with custom seed values, min/max ranges, and step increments.

You can also randomize the color attribute and custom float attributes. The existing falloff attribute, if enabled, will affect the amount of randomness applied, enabling spatially varying random effects.
