Beginner Tutorials

Start here if you're new to POPX or particle systems in TouchDesigner.

Tutorial 1: Your First Particle System

Beginner 15 minutes

Learn the fundamentals of POPX by creating a simple particle scatter system with rotation and color.

What You'll Learn:

  • Setting up POPX Scatter to emit particles
  • Connecting modifiers to transform particles
  • Rendering particles with Geometry COMP
  • Understanding the POPX attribute system

Network Setup:

  ↓
POPX Scatter (1000 particles)
  ↓

Tutorial 2: Using Falloffs for Spatial Control

Beginner 20 minutes

Understand how falloff fields work and use them to create localized particle effects.

  • Creating radial and geometry-based falloffs
  • Connecting falloffs to modifiers
  • Inverting and remapping falloff values
  • Combining multiple falloffs

Tutorial 3: Instancing Custom Geometry

Beginner 25 minutes

Replace particles with custom 3D models using TouchDesigner's instancing system.

What You'll Learn:

  • Setting up instance geometry in Geometry COMP
  • Using popxOrient for proper rotation
  • Scaling instances per-particle
  • Applying custom materials to instances
