Reporting Bugs

When reporting a bug, please include the following information:

  • POPX Version: Check the version number in the operator info panel
  • TouchDesigner Version: Build number from Help → About TouchDesigner
  • Operating System: Windows/macOS/Linux version
  • GPU Model: Graphics card and driver version
  • Steps to Reproduce: Detailed description of how to trigger the bug
  • Expected Behavior: What should happen
  • Actual Behavior: What actually happens
  • Screenshots/Video: Visual evidence if applicable
  • Error Messages: Copy from the TouchDesigner textport

Contributing

POPX is open source and we welcome contributions! Here's how you can help:

Code Contributions

  • Fork the repository and create a feature branch
  • Follow the coding standards outlined in CONTRIBUTING.md
  • Write tests for new features
  • Submit a pull request with a clear description

Documentation

  • Improve existing documentation pages
  • Write tutorials or guides
  • Translate documentation to other languages
  • Fix typos and broken links

Community Support

  • Answer questions on Discord and the forum
  • Share example projects and techniques
  • Create video tutorials
  • Write blog posts about POPX
